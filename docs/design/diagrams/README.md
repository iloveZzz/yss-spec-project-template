# 企业数据模型协作工作台图稿

本目录保存 Markdown 低保真原型的 Excalidraw 可视化补充材料。图稿只用于辅助评审和沟通，需求边界仍以 PRD、交互说明和后续 OpenAPI Draft 为准。

| 图稿 | 说明 | 来源 |
|---|---|---|
| `data-modeling-workbench-mvp-flow.excalidraw` | MVP 主流程图，覆盖项目进入、导入、建模、映射、检查、评审、发布、导出和回退路径。 | `docs/design/data-modeling-workbench-interaction-spec.md` 第 4 节 |
| `data-modeling-workbench-interaction-map.excalidraw` | 页面交互图，覆盖模型工作台、项目详情、导入向导、设计工作台、抽屉、评审页、发布弹窗和导出中心。 | `docs/design/data-modeling-workbench-interaction-spec.md` 第 3、5、7 节 |

打开方式：

1. 访问 `https://excalidraw.com/`。
2. 使用 Open 菜单导入对应 `.excalidraw` 文件。
3. 评审中发现的边界、状态或接口缺口，需要回写到 PRD、交互说明或 OpenAPI Draft，不能只停留在图稿里。
