# 发现阶段 (Discovery) — 市场分析 · 竞品分析 · 用户洞察

> **Phase 0 — 需求分析的前置输入。**

---

## 工作流

```
Ideation → 市场分析 → 竞品分析 → 用户洞察 → 商业论证 → 需求定义
```

---

## 市场分析

### 数据源
- 行业报告、上市公司财报、政府数据、投资机构研报

### 产出物
- 市场总览报告 / 规模与增长预测 / 趋势洞察

---

## 竞品分析

### 分析维度
产品功能 / 用户体验 / 技术能力 / 商业模式 / 用户口碑 / 市场表现

### 产出物
- 竞品功能矩阵 / 竞品深度分析 / 差异化机会清单

---

## 用户洞察

### 数据源
- 应用商店评论 / 社交媒体 / 客服社区 / 行业报告

### 产出物
- 用户画像 / 用户旅程地图 / 痛点优先级清单

---

## 商业论证

### AI Agent 任务
```python
delegate_task(
    goal="基于市场分析和竞品分析，生成商业论证草案",
    context="""
    输出：机会陈述 / 目标用户群 / 价值主张 / 商业模式 / 核心指标 / 风险评估
    """
)
```
